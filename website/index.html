Welcome to the Flood Prediction Project

This project provides real-time flood forecasts and environmental data.

About the Project

The Flood Prediction Project predicts flood risk based on sensor data, helping communities take timely precautions against potential flooding. It utilizes machine learning models to generate forecasts for the next 24 and 48 hours.

This project is available under the GNU General Public License Version 2 and can be accessed at the following link: GitHub Repository.

Developed by Adhyuth, Agnel, Adwayith, and Abhinav under the guidance of Dr. Jail Singh Bhokiya, this system aims to enhance early warning capabilities and disaster preparedness.
